# HR Analysis Project

An exploratory Human Resources (HR) data analysis project focused on understanding workforce patterns, employee behavior, and key factors affecting organizational decisions through data analysis and visualization.

---

## Project Overview

This project analyzes HR-related datasets to extract meaningful insights that can support data-driven decision-making in employee management and organizational strategy.

The analysis explores:

* Employee demographics
* Job roles and departments
* Salary distribution
* Employee performance indicators
* Attrition patterns
* Workforce trends
* Data visualizations and insights

The project is implemented using Jupyter Notebook and Python data analysis libraries.

---

## Objectives

The main goals of this project are:

* Clean and preprocess HR data
* Perform exploratory data analysis (EDA)
* Identify patterns and trends within employee data
* Generate visual insights
* Support HR decision-making through analytical findings

## How to Run the Project

### Clone the repository

```bash
git clone https://github.com/jannatul-fredaues/HR-analysis.git
```

### Move into the project directory

```bash
cd HR-analysis
```

### Install required libraries

```bash
pip install pandas numpy matplotlib seaborn notebook
```

### Start Jupyter Notebook

```bash
jupyter notebook
```

Open:

```bash
HR_analysis_corrected.ipynb
```
